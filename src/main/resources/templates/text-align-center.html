垂直和水平对齐文本中心

10 种不同的方式实现文本在容器中垂直和水平居中

方法 1: Flexbox (最常用)

使用 display: flex，配合 justify-content 和 align-items 实现完美居中

Flexbox 居中文本
.container {
display: flex;
justify-content: center; /* 水平居中 */
align-items: center; /* 垂直居中 */
}

方法 2: Grid (最简单)

使用 display: grid，配合 place-items 一行代码实现居中

Grid 居中文本
.container {
display: grid;
place-items: center; /* 同时处理水平和垂直 */
}

方法 3: Position + Transform

使用绝对定位配合 transform 实现居中

Position + Transform 居中
.container {
position: relative;
}
.text {
position: absolute;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);
}

方法 4: Table (老式方法)

使用 display: table 和 table-cell 实现居中

Table 居中文本
.container {
display: table;
width: 100%;
text-align: center;
}
.text {
display: table-cell;
vertical-align: middle;
}

方法 5: Line-height (仅单行文本)

设置 line-height 等于容器高度实现垂直居中，配合 text-align 水平居中

Line-height 单行居中
.container {
line-height: 200px; /* 等于容器高度 */
text-align: center;
}

方法 6: Absolute + Margin Auto

使用绝对定位和 margin: auto 实现居中

Margin Auto 居中
.container {
position: relative;
}
.text {
position: absolute;
top: 0;
bottom: 0;
left: 0;
right: 0;
margin: auto;
width: fit-content;
height: fit-content;
}

方法 7: Writing-mode (垂直文本)

使用 writing-mode 实现垂直文本对齐

垂直文本居中
.container {
writing-mode: vertical-rl;
text-orientation: mixed;
display: flex;
justify-content: center;
align-items: center;
}

方法 9: Grid 多行文本对齐

使用 Grid 对齐多行文本

Grid 多行文本
第一行
第二行

place-items: center;
text-align: center;
}

方法 10: Bootstrap 工具类

使用 Bootstrap 内置的工具类实现居中

Bootstrap 工具类居中
<!-- 直接在 HTML 中使用 Bootstrap 类 -->
<div class="d-flex justify-content-center align-items-center">
<!-- 内容 -->
</div>
